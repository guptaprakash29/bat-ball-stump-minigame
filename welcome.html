HTML

HTML (HyperText Markup Language) is standard markup laguage used to create Web pages. It provide the structure of the Web content and tells the browser how to display text, audio, video and other elements. It is one of the core technologies of the Web alongwith CSS (Style) and Javascript(Behaviour).

CSS

CSS (Cascading Style Sheet) is Style sheet language used to describe presentation of the documents that's written in HTMl/ XML. It is one of the core technologis of the Web alongwith HTMl(Structure) and Javascript(Behaviour).

JavaScript

JavaScript is HighLevel, Interpreted programming language that helps make Websites dynamic and Intercative. It is oe of the core technologis of the Web alongwith HTMl (Structure) and CSS(Style).

JAVA

JAVA is HighLevel, compiled, Strongly-Typed, Object Oriented Programming Language and Computing palteform developed by SunMicro System (now owned by Oracle) and originally designed by James Gosling.
